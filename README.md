# 🅿️ Free Parking Bot

Telegram-бот для поиска бесплатных парковок в Москве.

## Возможности

- 🔍 Поиск ближайших бесплатных парковок
- ➕ Добавление парковок пользователями
- 🗺️ Интерактивная карта (Telegram Web App)
- 📸 Просмотр панорам местности
- 🧭 Переход в Яндекс.Навигатор

## Установка

### 1. Клонирование/скачивание

```bash
cd "c:\Users\eriolnik\Desktop\Free Parking Bot"
```

### 2. Создание виртуального окружения

```bash
python -m venv venv
```

### 3. Активация виртуального окружения

**Windows:**
```bash
venv\Scripts\activate
```

**Linux/Mac:**
```bash
source venv/bin/activate
```

### 4. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 5. Настройка

1. Получите токен бота в [@BotFather](https://t.me/BotFather):
   - Создайте нового бота командой `/newbot`
   - Скопируйте полученный токен

2. Получите API-ключ Яндекс.Карт:
   - Зайдите в [кабинет разработчика](https://developer.tech.yandex.ru/)
   - Создайте новый ключ для **JavaScript API и HTTP Геокодера**
   - Включите бесплатные лимиты (до 25 000 запросов/день)

3. Создайте файл `.env` на основе `.env.example`:
   ```bash
   copy .env.example .env
   ```

4. Отредактируйте `.env`, вставив свои ключи:
   ```
   BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
   YANDEX_API_KEY=ваш-ключ-яндекс
   ```

## Запуск

```bash
python bot.py
```

## Использование

1. Откройте бота в Telegram
2. Нажмите `/start`
3. Отправьте свою геолокацию (кнопка 📎 → Геолокация)
4. Бот покажет ближайшие парковки
5. Выберите парковку и откройте навигатор
